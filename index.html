  1. Download nvim-linux64.tar.gz
  2. Extract: tar xzvf nvim-linux64.tar.gz
  3. Run ./nvim-linux64/bin/nvim

配置

  2. 在~/.config/nvim文件夹内创建init.lua文件

init.lua文件是neovim的核心文件，neovim会自动查看里面的配置来更新应用。

安装nerd字体

  • 需要特定的字体，比如Hack Nerd Font，才能支持nvim-web-devicons中的图标。

  • 或从webinstall.dev找到安装脚本

自动脚本下载的字体是：Droid Sans Mono for Powerline Nerd Font Complete.otf

  • 更改Terminal字体设置使之生效。在Windows Terminal中的设置步骤：

上部菜单中的设置→左侧菜单中的默认值和对应操作系统都设置→其他设置→外观→字体，选择安装的Nerd字体。
